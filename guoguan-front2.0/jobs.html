产品设计师（PD）

如果一个像素的误差让你寝室难安（像素眼），快到碗里来！

  1. 你是各类web2.0 产品的重度用户；
  2. 你经常无聊的时候就画个原型自己把玩；
  3. 你相信，设计可以改变世界；
  4. 你的美术功底，可以让你优雅的在手绘、点矢量ICON和界面设计之间无缝切换；
  5. 你对网站的简洁性和易用性有着偏执的追求（即便幻灯片这种控件也是多觉得是多余）；
  6. 你熟悉HTML5和CSS3规范及标准；
  7. 你熟悉ANDRIOD和IOS设计标准；

简历中想看到你的：

  1. 最骄傲的作品和详细的展示，以及以往作品或BLOG；
  2. 指出3个你常用的2.0产品中，让你爽的UE细节；
  3. 指出3个你常用的2.0产品中，最让你不爽的细节；
  4. 列出你工作中常用的工具，列出你最近在看的书籍；
  5. 你计划从广州到北京玩一周，列出整个过程中，你可能用到的互联网产品；

专题设计师(美术编辑)

如果你觉得BAT公司做的那些专题/宣传页/活动页/产品页设计的都很LOW，快到碗里来！

  1. 你会很多工具，用PHOTOSHOP就想吃饭睡觉一样自然；
  2. 你无聊时做一些有趣的设计；
  3. 你是女人们的宠物，磨皮特快，修片超赞；
  4. 你相信优秀的图片效果并非其本身的美观而是能唤醒人们内心的记忆；
  5. 你知道什么设计看起来很廉价；
  6. 你出入摄影棚，多屏操作一台MAC，一个硬盘的素材，手绘是你的强项，只要打开PS你就一定要拿出令人震惊的作品；
  7. 你定期为各种杂志供稿，或为某品牌产品做特效；
  8. 你熟悉web/app 设计理念；
  9. 你对文案也很擅长，烂的文案，通常你会选择拒绝设计；

简历中想看到你的：

  1. 3份以上你觉得能代表自己的作品，无需多加注释；
  2. 列举你定期翻阅的杂志、书籍、网站；
  3. 列举你认为做的很赞的效果及产品；
  4. 列举5个让你温暖的故事（可以来自身边/电影/小说……）

前端工程师

如果你一个人能干掉一个（产品经理+UED部门），快到碗里来！
如果你觉得前端是一份优雅和极富有创造力的工作，快到碗里来！

  1. 你真心看不上用网页三剑客的人；
  2. 你在git上开发或参与的项目，并持续维护；
  3. 你不会在不是多屏工作的公司上班；
  4. 你支持开源，对bootstrap 这种框架，喜爱程度是无法言语的；
  5. 你有很好的设计品味，坚持移动优先的理念，跨屏幕的响应式，有很多经验；
  6. 你是个超级APP控、工具控；
  7. 你精通HTML5和CSS3规范和标准；
  8. 你精通ANDRIOD和IOS开发标准；
  9. 你静通JS，手写是你的业余爱好；

简历中想看到你的：

  1. 一份以上你代表你技术最高造诣的前端作品；
  2. 列举几个你所熟知的前端大牛；
  3. 列举几个你觉得前端碉堡了的2.0产品，并加以介绍；
  4. 列出你工作中常用的工具；
  5. 你计划从广州到北京玩一周，列出整个过程中，你可能用到的互联网产品；

文案/策划/运营

如果你电脑最常用的动作是截图，快到碗里来！

  1. 你是收集癖吗？有一个硬盘装满了各种截图/文档/段子？；
  2. 你是女人们的闺蜜吗？外纯内腐，人艰不拆？
  3. 你是个网瘾症患者吗？饭局上朋友想剁了你的手？
  4. 你在哪几个社区里有5个以上的马甲？你有多少的粉丝？
  5. 你在500人的QQ群里刷屏聊天过吗？
  6. 你发表过那些让网友疯狂转载的段子/文章？
  7. 你离开电脑后，仍然满口的互联网词汇吗？
  8. 你能5秒钟内识别一条回帖是5毛所为么？
  9. 你很内涵吗？

如果以上条件符合3条以上，请继续往下看：

文案/策划

  1. 你有天马行空的创意，能快速的写出好段子/文章；
  2. 你有策划活动的能力并有方法让网友参与，引爆活动；
  3. 你和媒体关系不错，有持续供稿的渠道；
  4. 你是配图君，你是截图党，你是段子手，你……就是证据；
  5. 你有好品味，也许梦想和现实分不清，但不妨碍你做一个有趣的人；
  6. 你有广阔的知识体系，也能深入的研究，写出干而不涩的好内容；
  7. 你策划过很多专题，对互联网上传播的热点/新闻非常敏锐；

简历中想看到你的

  1. 2篇内容/1份专题/1场活动方案 3选2；
  2. 你的各种2.0社区的账号；
  3. 你最近在看的书，并附上自己的观点；
  4. 你电脑里的收藏；
  5. 你计划从广州到北京玩一周，列出整个过程中，你可能用到的互联网产品；
  6. 一则你生活中觉得有意思的事情；

运营

  1. 你有电商运营的经验（小米优先），并有独立策划/执行的营销案例；
  2. 你有社区/app运营的经验；
  3. 你有优质的流量渠道；
  4. 你能在10秒内识破哪些内容是公关文；
  5. 各大社区对你又爱又恨，和内部人员关系都很不错；
  6. 看过《乌合之众》

简历中想看到你的

  1. 2次以上的营销案例展示，有过程、效果、结果；
  2. 列举出你欣赏的互联网营销案例，并给出自己的观点；
  3. 说明你拥有哪些互联网资源；
  4. 列举出你觉得很牛的互联网运营人，并说出理由；
  5. 简短的分析话，互联网用户的大众行为。
  6. 列举出最近你看的好文章/好书。

商务（BD）

如果要办个事儿你脑子里下意识的是翻通讯录，快到碗里来！

  1. 你是个交际花，在互联网中有很多人脉关系；
  2. 你擅长资源整合，总能做出一些4两拨千斤的事情；
  3. 你有创业公司/BAT的工作经验，做过很多大型的活动；
  4. 你有资源互换的经验；
  5. 你有B2B深度合作的经验；
  6. 很多人追你
  7. 业余生活丰富

简历中想看到你的

  1. 1次以上资源整合的过程，结果；
  2. 展示你的人脉关系；
  3. 为我们想一份商务合作的方案；
  4. 你的生活照，社交账号;

总助

如果你是开心果，业务小能手，快到碗里来！

  1. 你是女性思维主导，为人处事温和而优雅，天生有一股吸引力、亲和力；
  2. 你对互联网的工作流程、性质、业务非常了解；
  3. 你对人事管理感兴趣，在校或在职期间，做过管理性质的岗位；
  4. 你外貌俊美，形象气质佳，每次出门，都会把自己打扮的清爽精致；

简历中想看到你的
